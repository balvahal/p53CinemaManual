FAQ

How do I get started / find help?
At the Matlab prompt, help panel, help panel/panel, help panel/demo or doc panel.
What's up with semilogx, semilogy, loglog, not producing log scales?
If you call select() on a new panel, it creates an axis for you. To protect font styling, it sets its "NextPlot" property to "replacechildren" (the default for new axes is "replace"). The high-level plotting functions semilogx, semilogy, loglog will not work correctly after this. Therefore, to use these functions with Panel, call the plotting function first and then select() the axis it creates into the panel. An example is given below.
>> figure; p = panel; >> semilogy(x,y,'r','linewidth',2.5); >> p.select(gca);
My dashed/dotted lines look tiny and/or grey and/or solid in an exported image file. Why?
This is a problem with the Matlab renderer, rather than with Panel itself. However, Panel can help - see demopanelI for details (or help panel/fixdash).
